## INSTALL
Install git, python3 and python3-pip!
```python
git clone https://github.com/miningseven/py_TgBot_BinanceFuture
cd py_TgBot_BinanceFuture
python3 -m pip install -r requirements.txt 
```
## RUN
setup your keys in settings.py
and run in terminal:
```python
python3 start.py
```
